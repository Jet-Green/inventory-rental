# Inventory Rental Frontend

Nuxt 4 + Vuetify клиент для сервиса аренды инвентаря.

## Что уже реализовано

- Базовый дизайн-слой в стиле `IWantATrip`: светлый фон, акцентный оранжевый, карточки с градиентом.
- Страницы:
  - `/` (лендинг + витрина),
  - `/rental-items` (каталог с фильтрами),
  - `/rental-item/[id]` (детали + бронирование + ссылки на PDF договоры).
- Архитектурный паттерн из `nmp`:
  - `app/api/*` для работы с backend,
  - `app/composables/*` для бизнес-логики и состояния через `useState`.
- `apiFetch` плагин и типизация доменных сущностей.

## Запуск

```bash
npm i
npm run dev
```

1. Скопируйте `.env.example` в `.env` (или используйте уже созданный `.env`).
2. Убедитесь, что `NUXT_PUBLIC_API_BASE` указывает на backend:
   - `http://localhost:4000/api`
3. Используйте локальную авторизацию через страницу `/auth`.

Текущее подключение к backend настраивается только через `.env`.
